## Рабочий процесс

- Ветка на задачу → PR в `main`. Прямых коммитов в `main` нет.
- CI гоняется на каждом PR: собираются обе прошивки. Деплой Pages и публикация
  Release с PR не запускаются — только с `main` и с тега.
- Мержим после зелёного CI.
- Стабильная точка → тег `v*`, который собирает Release с обоими образами.
