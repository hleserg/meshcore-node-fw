MeshCore Node FW

Companion-прошивки MeshCore, у которых companion-протокол выведен на аппаратный UART, а не на USB-CDC или BLE. WiFi и BLE отключены — нода подключается по проводу. Параметры радио (частота, BW, SF, CR) не зашиты в прошивку и задаются в рантайме.

Heltec WiFi LoRa 32 V4

ESP32-S3. Шьётся прямо из этой страницы по USB.

Браузер не поддерживает Web Serial. Откройте страницу в Chrome или Edge на десктопе. Web Serial работает только по HTTPS (или на localhost).

Или скачать образ вручную: v4-factory.bin (единый factory-образ: bootloader + partitions + приложение, пишется с offset 0).

Только Chrome или Edge Прошивка из браузера использует Web Serial API — он есть только в Chrome/Edge на десктопе и требует HTTPS. В Firefox и Safari кнопка работать не будет.

Heltec Mesh Node T114

nRF52840. Прошивается перетаскиванием файла .uf2 — из браузера не шьётся.

Скачать t114-companion.uf2

Как прошить

  1. Подключите T114 к компьютеру по USB.
  2. Быстро нажмите кнопку RST два раза.
  3. В системе появится диск T114BOOT (или похожий).
  4. Перетащите на него скачанный t114-companion.uf2.
  5. Плата перезагрузится сама — диск исчезнет. Готово.

Плата T114USB-UART / Flipper
GPIO9 (P0.09) — RXTX
GPIO10 (P0.10) — TXRX

Скорость 115200, 8N1. Уровни 3.3 В.

Web-прошивка для nRF52 невозможна T114 не отдаёт себя по Web Serial — единственный способ прошить его из браузера отсутствует физически. Только UF2-перетаскивание. Если после смены прошивки плата ведёт себя странно, разово прогоните erase-прошивку .uf2 и прошейте заново.
