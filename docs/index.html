Прошивка нод

Выбери плату и тип прошивки. V4 шьётся кликом прямо здесь, T114 — файлом. Ставить PlatformIO не нужно.

Нода сама выберет режим Companion-прошивка на старте читает detect-пин: провод от флиппера подключён — работает по проводу; ничего не подключено — поднимает WiFi-точку (V4) или BLE (T114). Смена режима = перезагрузка ноды.

Heltec V4 · companion

Воткни плату в USB и нажми кнопку. Нужен Chrome или Edge на десктопе.

Браузер не поддерживает Web Serial. Открой в Chrome или Edge на десктопе. Web Serial работает только по HTTPS.

Или скачать образ вручную: v4-companion-factory.bin (bootloader + partitions + приложение одним куском, пишется с offset 0).

Heltec T114 · companion

Скачать t114-companion.uf2
  1. Подключи T114 к компьютеру по USB.
  2. Быстро нажми RST два раза.
  3. Появится диск T114BOOT.
  4. Перетащи на него скачанный файл.
  5. Плата перезагрузится сама — диск исчезнет.

Heltec T114 · repeater

Стоковый ретранслятор: без автодетекта и без UART-патча — им это не нужно.

Скачать t114-repeater.uf2
  1. Подключи T114 к компьютеру по USB.
  2. Быстро нажми RST два раза.
  3. Появится диск T114BOOT.
  4. Перетащи на него скачанный файл.

Для V4 ретранслятора нет

Ретрансляторы в этой сети — только T114. Выбери T114 или вернись к companion.

После прошивки выставь радио Если настройки разойдутся хоть на йоту — ноды физически не услышат друг друга. 
meshcli -s COM5 set radio 869.525,250,11,5

freq 869.525 МГц · BW 250 кГц · SF 11 · CR 5. Порт свой. Можно и в мобильном приложении — там просто поля.
